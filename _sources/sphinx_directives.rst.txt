Блоки кода
==========

Блок кода с номерами строк:

.. code-block:: python
   :linenos:
   :emphasize-lines: 3,5

   def fibonacci(n):
       """Вычисляет n-е число Фибоначчи."""
       if n <= 1:
           return n
       return fibonacci(n-1) + fibonacci(n-2)
   
   # Пример использования
   print(fibonacci(10))

Включение файла кода:

.. literalinclude:: conf.py
   :language: python
   :lines: 1-10
   :caption: Первые 10 строк конфигурации

Глоссарий терминов
==================

Вы можете использовать термины из основного :doc:`glossary`:

* **API** - программный интерфейс приложения
* **Sphinx** - генератор документации  
* **Directive** - команда reStructuredText
